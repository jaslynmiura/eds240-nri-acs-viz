# Visualizing FEMA NRI Data (United States, 2025)

## About
This repository contains the HW2.qmd and HW3.qmd files, where a data visualizations is created to display:
- The FEMA National Risk Index scores for counties in California compared to other states in the United States.
- The FEMA National Risk Index ratings for various ethnic groups in California.
